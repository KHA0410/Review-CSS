Thuộc tính background-image

background-image: url(); -> có thể nối tiếp nhiều url để các hình chồng lên nhau url(), url(), ...
+ Có thể kết hợp vs linear-gradiant() để tạo 1 lớp màu gradiant phủ lên hình
background-image: url(), liner-gradiant(90deg, màu chuyển, màu chuyển); -> màu chuyển là rgba() để có độ trong suốt
Các thuộc tính khác khi đi cùng
width: 100%;
height: 100vh;
background-image: url();
background-size: 100px; // set cho width còn height auto
background-repeat: no-repeat;

Thuộc tính background-size với cover, contain

contain: kéo ảnh theo chiều ngang hoặc dọc có kích thước lớn hơn, không bị cắt xén ảnh
cover: ảnh sẽ bị mất khúc và đảm bảo luôn full màn

24/02 (64-70)

Thuộc tính background-origin

Thuộc tính background-origin sẽ giúp đối tượng hình ảnh đổ từ border, padding, content
background-origin: padding-box(content-box);

Background-position

background-position: top left; //trên cùng bên trái
background-position: top center; // trên cùng ở giữa
background-position: top 20px right 20px; //cách top 20px và right 20px
background-position: 100px 200px;
background-position: bottom; // mặc định hiểu tham số còn lại là center

Cú pháp shorthand của background

Viết tắt 4 thuộc tính background-image, backgound-repeat, background-position, background-size
background: url() no-repeat center/contain
Note: no-repeat viết sau url và giữa bg-position và bg-size phải có dấu "/"

Position (relative, absolute, fixed, sticky)

Relative(tương đối): lấy chính nó là góc tọa độ và có thể đè lên đối tượng khác

CSS{
posiotion: relative;
top: 100px;
left: 100px;
}

Absolute(tuyệt đối): phụ thuộc vào thẻ cha gần nhất có dùng thuộc tính position

CSS

.cha{
width: 500px;
height: 400px;
position: relative; //chỉ cần dùng posiotion là dc value gì cx dc
}
.con{
width: 100px;
height: 100px;
position: absolute;
top: 0;
right: 0;
}

=>box con nằm trong bên phải ở trên box cha

Fixed(dính): phụ thuộc vào cửa sổ trình duyệt, có thể ứng dụng thuộc tính này vào header, footer

CSS

header{
width: 100%;
height: 50px;
color: red;
position: fixed;
top: 0;
}

=>Lúc này header sẽ nhấc lên và nổi lên trên thẻ bên dưới sẽ chiếm chỗ của header, nên thẻ bên dưới phải thêm margin-top

Sticky(bám dính): kết hợp vs scroll để tạo hiệu ứng sroll header dính top

CSS

header{
width: 100%;
height: 50px;
color: red;
position: sticky;
top: 0;
}
